```markdown
# Security Policy

## Supported Versions
Only the latest stable branch architecture on the main track repository layout actively receives security path configurations or system patch optimizations.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | Yes (Current Main) |
| < 1.0.0 | No                 |

## Reporting a Vulnerability
Please do not file open bug tickets or raise visible issues if you identify an exploit or a credentials vector leak within our network handling code. 

Instead, report vulnerabilities privately by emailing the active developer maintainer team at `world_champ_dev@example.com` (or contact via your Lichess bot developer handling portal credentials). We aim to respond and evaluate reports within 48 hours.
```
